# MixxxHelpers
Add on controllers or tips for running Mixxx - a virtual DJ booth

[beatiItAutoDJ](./beatItAutoDJ/README.md) - Auto tempo and beat matching during AutoDJ transitioning, from current deck tempo to next deck so that the incoming deck plays at the original track tempo.  
[Sending a command to Mixxx](./remoteCommands/README.md) - How to remotely send a key press to Mixxx, allowing remote control of functions (simpler than via MIDI controllers, OSC or other RPC mechanisms).
